───────────────────────────────────────────────
        SHOP MODULE – FRONTEND → BACKEND → DB
───────────────────────────────────────────────
VUE FRONTEND (Shop Role)
│
├── ShopDashboard.vue
│     • Shows KPIs, charts, summary
│     • Calls:  GET /api/v1/shop/dashboard
│     └─→ shop_routes.py → reads Inventory, Sales models
│           └─→ SQLAlchemy → se_textile.db
│
├── ShopInventory.vue
│     • View/Add/Edit/Delete stock
│     • Calls:
│          GET  /api/v1/inventory/all
│          POST /api/v1/inventory/add
│          PUT  /api/v1/inventory/update/<id>
│          DEL  /api/v1/inventory/delete/<id>
│     └─→ inventory_routes.py → Inventory model
│           └─→ db.session CRUD on Inventory table
│
├── ShopInquiry.vue
│     • Send or view customer inquiries
│     • Calls:
│          POST /api/v1/inquiry/create
│          GET  /api/v1/inquiry/all
│     └─→ inquiry_routes.py → Inquiry model
│           └─→ db.session add/read Inquiry
│
└── ShopMarketing.vue
      • Manage offers and campaigns
      • Calls:
           GET  /api/v1/marketing/data
           POST /api/v1/marketing/add
      └─→ marketing_routes.py → Marketing model
            └─→ db.session CRUD on Marketing table
───────────────────────────────────────────────
COMMON FLOW
───────────────────────────────────────────────
[User Clicks Button on Vue Page]
        │
        ▼
Axios (src/api/*.js) → HTTP request
        │
        ▼
Flask Route (routes/*.py) receives request
        │
        ▼
Business Logic / Validation
        │
        ▼
SQLAlchemy ORM ↔ Database
        │
        ▼
Flask sends JSON response
        │
        ▼
Axios receives → Vue updates UI dynamically
